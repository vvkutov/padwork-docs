Применение искусственного интеллекта поддерживается для следующих шаблонов досок:
- Стена
- Блог
- Персоны

Доступно 2 функции с искусственным интеллектом. Нажмите на заголовок, чтобы узнать подробнее:
1. [Генерация контента](/ai/content-generation) - генерация карточек с помощью искусственного интеллекта. Например, вы можете задать запрос "советы по резюме" для генерации советов по резюме.
2. [Спроси ИИ](/ai/ask-ai) - возможность задать вопрос по информации с доски. Данная функция становится доступна когда у вас на доске есть 3 и более карточек с текстом.
