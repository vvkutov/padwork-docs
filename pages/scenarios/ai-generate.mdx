# Сценарий "Генерация контента (ИИ)"

Цель: Создать учебный контент с использованием искусственного интеллекта для повышения эффективности и разнообразия материалов.

Описание: Преподаватель формулирует тему или задачу, которую необходимо осветить в учебном материале. Затем он использует ИИ для генерации контента. ИИ анализирует запрос и генерирует новый контент (например: планы занятий, советы, примеры, упражнения, тесты и другие материалы), который преподаватель может использовать в процессе обучения.



## Шаги
1. В разделе "Доски" создайте новую доску, укажите следующие данные:
     - **Шаблон**: Стена
     - **Формат карточек на стене**: вертикальный
     - **Количество карточек в строке**: 4
2. Нажмите __Далее__
3. Заполните поля:
     - **Название**: Название группы
     - **Описание**: Дисциплина
     - **Тип доски**: Приватная
4. Нажмите __Сохранить__
5. На доске нажмите на кнопку "Задание", заполните поля:
    - **Текст задания** - что нужно сделать
    - **Доп. требования к заданию (что должен проверить ИИ)** - выпишите требования к ответу на задание. Например: "в ответе должно быть не менее 4 пунктов", или "в ответе должно быть минимум 3 пункта, каждый должен быть описан". **Студенты видят данное требование**
    - **Минимальная оценка** и **Максимальная оценка** - количество баллов для оценки (например, от 0 до 5)
    - __Ответ на задание писать не нужно, ИИ проверяет информацию самостоятельно__
6. На доске нажмите на кнопку __Генератор контента__
7. Напишите запрос. Например:
    - Сгенерируй советы по составлению резюме
    - Напиши план занятия по дисциплине __...__
    - Сгенерируй тест из 10 вопросов по дисциплине __...__ на тему __...__
